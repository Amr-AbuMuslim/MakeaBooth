# Transformers Display System

A premium, modern website for Transformers - eco-friendly, foldable aluminum display systems for exhibitions, trade shows, and events.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations** - Framer Motion animations for engaging user experience
- **Interactive FAQ** - Accordion-style FAQ section with smooth transitions
- **Product Showcase** - Comprehensive gallery of booth types and accessories
- **Contact Management** - Easy-to-use contact form with booth type selection
- **SEO Optimized** - Built with Next.js for optimal search engine visibility

## Tech Stack

- **React.js 18** - React Library with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **shadcn/ui** - High-quality React components
- **Lucide React** - Beautiful icon library

## Installation

### Option 1: Using shadcn CLI (Recommended)

1. Clone the repository:
   \`\`\`bash
   git clone <repository-url>
   cd transformers-display-system
   \`\`\`

2. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`

3. Run the development server:
   \`\`\`bash
   npm run dev
   \`\`\`

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

### Colors
Edit the design tokens in `app/globals.css`:
- Primary: Navy (#0A2540)
- Secondary: Gold (#D4AF37)
- Accent: Aqua (#00BCD4)

### Fonts
Fonts are configured in `app/layout.tsx`:
- Heading: Playfair Display
- Body: Inter

### Content
Update page content directly in:
- Home: `app/page.tsx`
- About: `app/about/page.tsx`
- Products: `app/products/page.tsx`
- FAQ: `app/faq/page.tsx`
- Contact: `app/contact/page.tsx`
